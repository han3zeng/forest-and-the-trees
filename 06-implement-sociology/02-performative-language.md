# Performative Language
* 以社會學角度看說話，說話也是一直在改變社會關係的

## Overview
* 表演式語言
    * 改變社會結構（關係），會有社會後果
        * examples
            * 我愛你
            * 我發誓
            * 婚姻誓約（我願意）
            * 我保證
            * 我要離職
            * 我很抱歉
    * 有分強弱
        * 你很迷人效果與我愛你不同
    * 我經由社會文化，來架構「表演式語言」的結果
        * 我愛你
            * 我愛你的互惠模式（要回答我也愛你）
                * 期待得到「我愛你回饋」
            * 對特定人才會講
            * 不同人對於我愛你的定義不一樣
                * 父母親與子女
            * 伴侶會以我愛你的互惠最為一種關係連結的保證
            * 告白時，我愛你即是希望一種關係的改變
